Lenovo TB-X605F 超详细刷机救砖教程

前言

正常的刷机操作并不会损坏手机硬件，但不当的刷机方法可能会带来不必要的麻烦
使用不当的刷机方法可能会导致包括但不限于以下问题:

无限重启
无限Fastboot
无限Recovery
开机弹到900E而不是9008

本教程的刷机包仅适用于搭载SDM450处理器的TB-X605F 与搭载SDA429处理器版本的设备不兼容
该文章使用的是9008刷机方式,请提前备份重要数据

准备工作

QFIL刷机工具 或 MiFlash

QFIL

QFIL下载链接(如无法访问请使用VPN): QFIL_Repacked.zip
下载完成后将QFIL_Repacked.zip解压以便后续操作

不要用官网的QFIL，否则会出现如下情况:

MiFlash

MiFlash下载链接:MiFlash2020-3-14-0.rar

MD5校验工具

Md5CheckerCn.zip

线刷包

下载链接: TB-X605F_S210224_201119_ROW.zip

数据线

准备原装数据线，如果您的原装数据线丢失或损坏请使用其他可以进行稳定的数据传输的数据线

镊子或其他金属物品

用于短接9008触点

开始刷机

对刷机工具 及 刷机包 进行MD5校验

MD5:

1.定位到解压缩后的MD5校验工具文件夹

2.打开Md5CheckerCn.exe

随后出现如下窗口

3.将QFIL_Repacked.zip 或 MiFlash2020-3-14-0.rar拖入窗口中

4.右键Md5CheckerCn中的QFIL_Repacked.zip 或 MiFlash2020-3-14-0.rar

单击比较 MD5 值
出现如下窗口

5.将MD5值粘贴进输入框中，出现”相同”即为文件完整


如果文件完整则可以进行后面的步骤

如出现”不同”请重新下载文件（如网络环境不好避免使用IDM下载）

使用 QFIL刷机工具 或 MiFlash 进行刷机

通过QFIL刷机

1.打开QFIL_Repacked文件夹

2.右键QFIL.exe -> 以管理员身份运行

3.在弹出的窗口中单击”是”

随后出现如下窗口

4.选择 “Flat Build” 单选框

5.单击 “Browse”

在弹出的窗口中定位到解压缩后的刷机包文件夹

6.双击 “prog_emmc_firehose_8953_ddr.mbn” 选择Programmer

7.单击 “Load XML…”

在第一次弹出的窗口中定位到解压缩后的刷机包文件夹
双击 “rawprogram_unsparse_upgrade.xml”

在第二次弹出的窗口中定位到解压缩后的刷机包文件夹
双击 “patch0.xml”

8.将平板关机，短接9008触点

如果平板目前处于900E状态，请在短接触点的同时长按电源键直到设备管理器出现9008设备

9008触点图↓

9.单击 “Select Port…”

单击列表框中的9008设备

单击OK

单击Download

等待读条

读条完成后拔出数据线，长按电源键直到 “Lenovo” Logo出现

至此，通过QFIL进行刷机救砖成功

通过MiFlash刷机

1.打开MiFlash2020-3-14-0文件夹

2.右键XiaoMiFlash.exe -> 以管理员身份运行

3.在弹出的窗口中单击”是”
随后出现如下窗口

4.单击左上角的 “Configuration”
5.单击 “MiFlash Configuration”
出现如下界面

6.单击下方第一个 “select”
在弹出的窗口中定位到解压缩后的刷机包文件夹
双击 “rawprogram_unsparse_upgrade.xml”

7.单击下方第二个 “select”
在弹出的窗口中定位到解压缩后的刷机包文件夹
双击 “patch0.xml”

8.单击 “Main program” 旁边的箭头，选择 “fh_loader”

9.单击 “OK”

10.回到MiFlash主页面

单击 “选择”
在弹出的窗口中定位到解压缩后的刷机包文件夹
单击”OK”

11.单击 “加载设备”

12.列表中出现设备后单击 “刷机”

13.等待读条完成长按电源键直到 “Lenovo” Logo出现

至此，通过MiFlash进行刷机救砖成功
